女高中生的虚度日常

发表于 2020-02-28 | 分类于 搞笑日常番

原始作品：漫画改编
动画导演：高桥丈夫（总），三瓶圣
背景音乐：菊谷知树
动画脚本：横谷昌宏
角色设计：安田祥子
动画集数：12集
作品风格：日常|校园|搞笑|青春
制作公司：-（Passione）
发行日期：2019年夏季番

主要人物

田中望
配音：赤崎千夏
性格脱线的笨蛋，本作女主角之一，外貌和成绩都处于下等，开学第一天就给班上的所有同学取了外号，说话不会看场面，是死宅开始画漫画的动力之一。





菊池茜
配音：户松遥
喜欢漫画的御宅族少女，未来想成为漫画家。擅长打扮和服装搭配。 非常崇拜独立音乐制作人低所得P，翻唱过他的歌曲，在现场见面会发现他就是自己的班主任“早稻田”（佐渡正敬），被他的话语激励，虽然感到很羞耻，但是还是喜欢他，听了他的新曲后，决定继续坚持自己的梦想，并有一天要追上他。






百井咲久
配音：长绳麻理亚
身材娇小的可爱少女，但对自己的偏幼的容貌和身材非常自卑，因此装作一副太妹的样子。因为父母远赴德国，自己和奶奶生活在一起，其实本质上是个天真可爱的小孩子。





山本美波
配音：富田美忧
身上缠有绷带来掩盖青春痘、痱子，也为了耍酷。平时常在校服外套一件黑色外套，染着一头金发，是个十足的中二病。






主要参数

恋爱：★★★★☆
搞笑：★★★★★
画面：★★★☆
剧情：★★★☆
观感：★★★★☆

吐槽（以下内容可能含剧透）

挺搞笑的一部日常动漫，说日常的其实有很多，但讲女子高中生的日常却是比较少，本作将JK的有趣而普通的生活展现出来了，至少我看得乐了。

角色类型较多，中二病属性，萝莉属性，百合属性，少言闷骚，腐女等等，每个角色都有自己的特点，不得不说基本囊括了全部的女子高中生了……

剧情方面是日常就没什么可说的了，制作表现来看，中规中矩，但也是一部可看的好番剧。

评分：9.41
